## 4. CI/CD集成

### 4.1 基础集成

在CI中运行验证：

```yaml
# .github/workflows/sdd-validate.yml
name: SDD Validate

on:
  pull_request:
    paths:
      - 'features/**'

jobs:
  validate:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - name: Run validation
        run: |
          ./scripts/auto-workflow.sh ${{ matrix.feature }} single validate
```

### 4.2 自动合并

验证通过后自动合并：

```yaml
name: Auto Merge

on:
  workflow_run:
    workflows: [SDD Validate]
    types: [completed]
    branches: [main]

jobs:
  merge:
    if: ${{ github.event.workflow_run.conclusion == 'success' }}
    runs-on: ubuntu-latest
    steps:
      - name: Merge
        run: |
          gh pr merge --auto $PR_NUMBER
```

## 5. 通知集成

### 5.1 Slack通知

阶段完成时通知：

```bash
# 在after_artifact_write hook中
if [ -n "$SLACK_WEBHOOK" ]; then
    curl -X POST -H 'Content-type: application/json' \
        --data '{"text":"Stage completed: '$STAGE' for '$FEATURE'"}' \
        $SLACK_WEBHOOK
fi
```

### 5.2 邮件通知

审批请求时发送邮件：

```bash
# 在创建检查点时
echo "检查点需要审批: $FEATURE" | \
    mail -s "SDD检查点" $EMAIL
```
